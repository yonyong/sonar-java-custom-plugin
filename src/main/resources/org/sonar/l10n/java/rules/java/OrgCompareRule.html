年轻人，你trim()了吗

orgCode比较用trim方法

class MyClass {
    boolean NonCompliantEquals(OrgInfo orgInfo){ // Noncompliant
        return "NJ001".equals(orgInfo.getOrgCode());
    }
}

Compliant Solution

class MyClass {
    boolean CompliantEquals(OrgInfo orgInfo){ // Compliant
        return "NJ001".equals(orgInfo.getOrgCode().trim());
    }
}
